An example chatbot using [Gradio](https://gradio.app), [`huggingface_hub`](https://huggingface.co/docs/huggingface_hub/v0.22.2/en/index), and the [Hugging Face Inference API](https://huggingface.co/docs/api-inference/index).
This project is a web-based chatbot application designed to assist users with yoga-related queries. The chatbot can provide information on class schedules, instructor info, registration, membership plans, studio guidelines, yoga tips, and other related questions.
